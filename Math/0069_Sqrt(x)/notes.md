# Sqrt(x)

## Difficulty

Easy

## Pattern

Math 

## Concepts Used

- Math module
- Sqrt and floor function

## Approach

Import math module.

- Use math.sqrt(x) function to find sqrt.
- Using math.floor function round it to the nearest integer.
- Use int() function to convert it to the integer.

## Time Complexity

Overall = **O(n)**

## Space Complexity

**O(1)**
