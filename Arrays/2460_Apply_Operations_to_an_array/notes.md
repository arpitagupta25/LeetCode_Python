# Apply Operations to an Array

## Difficulty
Easy

## Pattern
Two-pointer concept

## Concepts Used

- Array
- Two-pointer concept
- Duplicate Handling 

## Approach

Use two-pointer technique.
Visit every adjacent pair.
If both numbers are the same:
Double the first number.
Make the second number 0.
Return L

## Time Complexity

Overall = O(n)

## Space Complexity

O(1)

## Mistakes I Made

Moving zeros before applying operations
Only increment write when you place a non-zero element.
