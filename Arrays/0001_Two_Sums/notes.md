# Two Sum

## Difficulty
Easy

## Pattern
Hash Map, Arrays

## Concepts Used

- Dictionary
- One-pass traversal

## Approach

Store previously visited numbers.
For every element:
Check if target-current exists.
Otherwise store current element.

## Time Complexity

O(n)

## Space Complexity

O(n)

## Mistakes I Made

Forgot duplicate values.
Initially solved using nested loops.
